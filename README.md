# 지역별 사진 자동 분류기 
>**여행때 찍은 사진들을 지역별로 묶어서 볼 수 있어요!**



사진 파일에 저장된 **GPS** 메타데이터 및 **EXIF** 를 추출하여 사진이 촬영된 지역별로 폴더를 생성하고 자동 분류해 주는 파이썬 프로그램입니다. **Pillow** 와 **geopy** 라이브러리 를 활용하여 좌표 추출 및 행정 구역명 변환을 수행합니다.

## 주요 기능

- 사진 내 **EXIF** 데이터 에서 위도와 경도 좌표 추출
- **geopy** (Nominatim API) 를 이용한 역지오코딩(Reverse Geocoding)으로 좌표를 한국어 지역명(시/도/군 등)으로 변환
- 지역별 폴더 자동 생성 및 사진 복사 (원본 유지)
- **GPS** 정보 가 없는 사진은 따로 분류하여 관리

## 요구 사항

이 프로그램을 실행하려면 다음 라이브러리가 필요합니다.

- Python 3.x
- Pillow
- geopy

## 설치 방법

터미널이나 명령 프롬프트에서 아래 명령어를 실행하여 필수 라이브러리를 설치하세요.

```
pip install Pillow geopy

```

## 사용 방법

1. `sortphoto.py` 파일의 설정 부분을 본인의 환경에 맞게 변경합니다.

```python
# 설정 부분
SOURCE_FOLDER = './my_photos'  # 정리할 사진들이 있는 원본 폴더 경로
DEST_FOLDER = './sorted_photos' # 지역별로 분류된 사진이 저장될 새 폴더 경로

```

2. 분류하고자 하는 사진들을 **SOURCE_FOLDER** 에 지정한 경로에 넣습니다.
3. 프로그램을 실행합니다.

```bash
python sort_photos.py

```

## 프로젝트 구조

```text
├── sort_photos.py
├── my_photos/          # 원본 사진 폴더 (작업 전)
└── sorted_photos/      # 분류 완료된 폴더 (작업 후)
    ├── 서울특별시/
    ├── 제주특별자치도/
    └── GPS_정보_없음/
```
